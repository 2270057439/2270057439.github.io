仅仅需要一个platform-tools文件夹

方法一：因为安卓开发环境需要用到adb，安装过studio等软件的都自带的有platform-tools，它在~/Library/Android/sdk/platform-tools下，
方法二：简单粗暴，官网下载,下载platform-tools文件夹。
方法三：先保证有brew包管理器，再执行brew cask install android-platform-tools即可。
2.配置环境变量
为方便adb的使用，添加环境变量。
iTerm输入open -e .bash_profile,打开配置文件。
添加export PATH=${PATH}:~/Library/Android/sdk/platform-tools即可。

一：换源篇。

shell换源

众所周知，使用shell第一步就是换源，毕竟祖国有一面伟大的墙。此篇适用于各种shell（实测zsh，bash以及arm平台的termux）

pip换源。

1.查看根目录是否有.pip文件夹，没有就新建一个，然后在.pip 文件夹内新建一个pip.conf文件。

ls -a //查看
mkdir .pip//新建文件夹
touch pip.conf//新建文件

2.粘贴代码到pip.conf

//阿里
[global]
index-url = http://mirrors.aliyun.com/pypi/simple/
[install]
trusted-host=mirrors.aliyun.com

//豆瓣
[global]
index-url = http://pypi.douban.com/simple
[install]
trusted-host=pypi.douban.com

//清华（推荐）
[global]
index-url = https://pypi.tuna.tsinghua.edu.cn/simple
[install]
trusted-host=pypi.tuna.tsinghua.edu.cn

brew换源

// 替换brew.git:
$ cd "$(brew --repo)"

// 中国科大:
$ git remote set-url origin https://mirrors.ustc.edu.cn/brew.git
// 清华大学:
$ git remote set-url origin https://mirrors.tuna.tsinghua.edu.cn/git/homebrew/brew.git



// 替换homebrew-core.git:
$ cd "$(brew --repo)/Library/Taps/homebrew/homebrew-core"
// 中国科大:
$ git remote set-url origin https://mirrors.ustc.edu.cn/homebrew-core.git
// 清华大学:
 git remote set-url origin https://mirrors.tuna.tsinghua.edu.cn/git/homebrew/homebrew-core.git


// 替换homebrew-bottles:
// 中国科大:
$ echo 'export HOMEBREW_BOTTLE_DOMAIN=https://mirrors.ustc.edu.cn/homebrew-bottles' >> ~/.bash_profile
$ source ~/.bash_profile
// 清华大学:
$ echo 'export HOMEBREW_BOTTLE_DOMAIN=https://mirrors.tuna.tsinghua.edu.cn/homebrew-bottles' >> ~/.bash_profile
$ source ~/.bash_profile


// 应用生效:
$ brew update

二：报错篇

python3
报错：ModuleNotFoundError: No module named ‘PIL’
解决方法
运行安装pillow命令:

pip install pillow
//如果运行时显示Requirement already satisfied: pillow in /anaconda3/lib/python3.7/site-packages (5.4.1)
//表示已经安装过了，这时可以先卸载，获取最新的pillow
//运行卸载命令: 
pip uninstall pillow
//然后再次运行安装命令
pip install pillow
